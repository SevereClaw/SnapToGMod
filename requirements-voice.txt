# Дополнительные зависимости ТОЛЬКО для голосового выбора персонажа
# (voice_select.py). Без них программа работает как обычно, просто пункт
# «Голосовой выбор персонажа» в трее будет показывать, каких библиотек не
# хватает, и не включится.
#
# Установка:
#   pip install -r requirements.txt
#   pip install -r requirements-voice.txt
#
# Также нужно (не через pip):
#   1) Скачать и распаковать модель Vosk (например, vosk-model-small-ru-0.22)
#      в папку %APPDATA%\SnapToGMod\voice\model — пункт меню трея
#      «Настройки -> Голосовой выбор персонажа -> Открыть папку модели/шаблонов»
#      откроет нужное место.
#   2) Положить туда же, в %APPDATA%\SnapToGMod\voice\templates, три
#      скриншота-обрезка из игры:
#        menu_choose_character.png  — пункт меню паузы "Выбрать персонажа"
#        nagito_icon.png            — иконка персонажа в разделе избранного
#        select_button.png          — кнопка "ВЫБРАТЬ ПЕРСОНАЖА"
vosk
pyautogui
pydirectinput
pygetwindow
opencv-python
